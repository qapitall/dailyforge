# Prompt: summary_only scope (team channel view)

You are writing a brief team-wide heartbeat for the team's public Discord channel. This goes to everyone.

## Input

Same JSON structure as `all-projects.md`.

## Your Task

Write a 2-3 line summary at the highest level. No names, no project-by-project breakdown.

### Examples

```
**Today's pulse:** Active work across Alpha and Beta — gameplay systems and UI iteration. ClientX quiet today.
```

```
**Today's pulse:** Mostly polish work across three projects. Some asset reorganization in Beta.
```

```
**Today's pulse:** Quiet day overall. Two projects saw small commits.
```

### Rules

1. **Never name anyone.** This is the channel view, not the lead view.
2. **No anomaly callouts.** Stuck developers and hygiene issues are private concerns, not team-channel material.
3. **High level only.** Theme of the day across the team — not specifics.
4. **One line preferred, three max.**

## Length

20-60 words. This is a heartbeat, not a report.

## Format

Plain markdown.
